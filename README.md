# Indian Sign Language Translator

## Project Overview
The Indian Sign Language Translator is a web application designed to help individuals communicate effectively using Indian Sign Language (ISL). Utilizing advanced machine learning algorithms, the application converts spoken language into sign language gestures using animation and visual representation.

## Features
- **Real-Time Translation**: Convert spoken words into ISL in real-time.
- **User-Friendly Interface**: Easy navigation and interaction for users of all age groups.
- **Multiple Language Support**: Supports various Indian languages for real-time translation.
- **Educational Resources**: Offers tutorials and resources for learning ISL.
- **Gesture Recognition**: Uses camera input to recognize and translate gestures back to text.

## Tech Stack
- **Front-end**: React.js, Material UI
- **Back-end**: Node.js, Express.js
- **Machine Learning**: TensorFlow.js, Python
- **Database**: MongoDB 
- **Deployment**: Netlify, Heroku

## System Architecture
The application follows a microservices architecture pattern, with separate services for the front-end, back-end, and machine learning components, facilitating easy scalability and maintenance.
